## 📖 Overview

BudgetBuddy is a next-generation personal finance platform that combines traditional budgeting with blockchain technology and AI-powered financial guidance.

Instead of only tracking expenses, BudgetBuddy gives users a complete financial dashboard where they can:

- Monitor cryptocurrency wallets
- Track income and expenses
- View financial analytics
- Receive AI-generated budgeting suggestions
- Manage multiple accounts in one place

The goal is to simplify personal finance while introducing secure decentralized technologies.
